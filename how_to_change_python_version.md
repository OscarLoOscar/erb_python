<h2>3 Method to change python version</h2>

**1.global**

```bash
pyenv global 3.12.12
```
每次開機都係呢個version

---
**2.local**

```bash
pyenv local 3.12.12
```
只有呢個dir先係呢個version

---
**3.shell**

```bash
pyenv shell 3.14.2
```

--- 
**Create Virtual Environment**

```bash 
mkvirtualenv erb8
```

---
check python version

```bash
python -V
```
